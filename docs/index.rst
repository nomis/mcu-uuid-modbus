mcu-uuid-modbus
===============

Description
-----------

Microcontroller asynchronous `Modbus`_ library

Purpose
-------

Provides an asynchronous client for communicating with `Modbus`_ devices. This
library is for single threaded applications and cannot be used from an interrupt
context.

Dependencies
------------

* `mcu-uuid-common <https://mcu-uuid-common.readthedocs.io/>`_
* `mcu-uuid-log <https://mcu-uuid-log.readthedocs.io/>`_

Refer to the :download:`library.json <../library.json>` file for more
details.
